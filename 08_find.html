Array.prototype.find()
        시도해보기
        find() 메서드는 
        조건 결과를 새로운 값 반환합니다.

        구문
        arr.find(callback(element[, index[, array]])[, thisArg])
        arr.find(function( 현재값 ) => {
            return  조건 
        })
        arr.find( 현재값 => 조건  )

        매개변수
        callback : 함수 
        element :  현재값 
        index : 인덱스.
        
        반환 값
        조건 맞는  새로운 하나의값.
